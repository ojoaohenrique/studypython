# Python

## 📖 O que é Python?

Python é uma linguagem de programação popular criada por **Guido van Rossum** e lançada em **1991**.

É uma linguagem versátil, amplamente utilizada em diversas áreas da tecnologia, desde desenvolvimento web até inteligência artificial e análise de dados.

---

## 🚀 Principais Aplicações

Python pode ser utilizado para:

- Desenvolvimento web (backend)
- Desenvolvimento de software
- Automação de tarefas
- Scripts de sistema
- Matemática e computação científica
- Ciência de dados e Big Data
- Inteligência Artificial e Machine Learning
- Manipulação de arquivos e bancos de dados

---

## ⚙️ O que o Python pode fazer?

- Criar aplicações web executadas em servidores
- Automatizar fluxos de trabalho e processos
- Conectar-se a bancos de dados
- Ler, criar e modificar arquivos
- Processar grandes volumes de dados
- Realizar cálculos matemáticos complexos
- Desenvolver protótipos rapidamente
- Construir sistemas prontos para produção

---

## ⭐ Por que utilizar Python?

Python é uma das linguagens mais populares do mundo devido às seguintes características:

- Funciona em múltiplas plataformas (Windows, Linux, macOS e Raspberry Pi)
- Sintaxe simples e de fácil compreensão
- Exige menos linhas de código que muitas outras linguagens
- Permite desenvolvimento rápido de aplicações
- Possui grande quantidade de bibliotecas e frameworks
- Suporta diferentes paradigmas de programação

---

## 🧩 Paradigmas de Programação

Python pode ser utilizado de diferentes formas:

- Programação Procedural
- Programação Orientada a Objetos (POO)
- Programação Funcional

## 📝 Sintaxe do Python

Python foi projetado para ser uma linguagem legível e fácil de aprender.

### Exemplo:

```python
print("Olá, Mundo!")
```

### Saída:

```text
Olá, Mundo!
```

---

## 🔍 Diferenças em relação a outras linguagens

Python possui algumas características que o tornam único:

- Utiliza quebras de linha para finalizar comandos
- Não exige ponto e vírgula (`;`) ao final das linhas
- Utiliza indentação para definir blocos de código
- Prioriza legibilidade e simplicidade

### Exemplo de Indentação

```python
idade = 18

if idade >= 18:
    print("Maior de idade")
```

A indentação define o escopo de estruturas como:

- Condições (`if`)
- Laços de repetição (`for`, `while`)
- Funções
- Classes
